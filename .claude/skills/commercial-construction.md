# Commercial Construction Skill

How a commercial project actually goes from contract to keys-in-hand.
Apply when planning phases, organizing the construction process, writing
or reviewing CDs, or interpreting field conditions.

---

## Project delivery methods

| Method | Sequence | Risk | When to use |
|---|---|---|---|
| **Design-Bid-Build (DBB)** | Design complete → bid → construct | Owner / Architect carry design risk; Contractor carries means/methods | Most public work; clear scope, time available |
| **Construction Manager at Risk (CMAR / CM/GC)** | CM hired at SD; gives GMP at DD/CD | Shared; CM commits to GMP | Complex projects; tight budget control; want builder input |
| **Design-Build (DB)** | Owner contracts one entity (DB) that designs + builds | DB carries both | Tight schedule; performance-spec building; fewer variables |
| **Integrated Project Delivery (IPD)** | Owner + Architect + Contractor share contract | Shared via multi-party agreement | High collaboration; sophisticated owner |

90% of commercial projects use DBB or CMAR. Know both.

## The phases

### Programming
- Owner needs assessment → BOD (Basis of Design) narrative
- Spaces: NSF, GSF, departmental adjacencies, headcount, growth
- Site analysis: zoning, utilities, soil
- Cost ROM (Rough Order of Magnitude): ± 30%
- Schedule ROM
- Deliverable: Program document signed by owner

### Schematic Design (SD)
- Site plan, floor plans, key elevations, building section
- Structural grid set; floor-to-floor heights
- Mechanical system type selected
- Code analysis sketch
- Cost: ± 15%
- Deliverable: SD set, design narrative, code analysis sketch

### Design Development (DD)
- Drawings developed: plans, elevations, sections, key details
- Wall types, door / window types, finishes selected
- Structural sized; MEP routing; coordinated
- Specifications outline (sections list)
- Cost: ± 10%
- Deliverable: DD set, specs outline, code analysis narrative

### Construction Documents (CD)
- Complete drawing set: all sheets, all schedules, all details
- Complete specifications (3-part format, Division 00-49)
- Stamped by Architect + Engineers
- Submitted for permit
- Cost: ± 5% (bid documents)
- Deliverable: CD set, project manual, permit submission

### Bidding / Negotiation (B/N)
- Issue for bid → addenda → bid opening
- Or in CMAR: GMP negotiation
- Contractor selected; AIA contract signed
- Notice to Proceed (NTP)

### Construction Administration (CA)
- Submittals: shop drawings, product data, samples, mockups
- RFIs (Request for Information): contractor questions
- ASIs (Architect Supplemental Instructions): clarifications
- COs (Change Orders): scope/cost/time changes
- Field reports: site visit observations
- Pay applications: monthly draws
- Substantial completion (S.C.): owner can occupy
- Punch list: items to complete
- Final completion: 1-year warranty starts
- Deliverable: as-built drawings, O&M manuals

## RFI process

Standard timeline:
- Contractor submits RFI → Architect logs in RFI tracker
- Routed to discipline (Arch / Struct / MEP / Civil)
- Response within 5-10 business days (per contract)
- Response distributed to all CMs / subs
- Logged with sequential number, status, response date

Common RFI categories:
- Coordination conflict (M+E in same chase)
- Missing detail (jamb at unique condition)
- Substitution request (different product than specified)
- Field condition different from drawings

The lower the RFI count, the more complete the CDs.

## Submittal process

Three levels:
- **Shop drawings** — fabricator's drawings (steel, curtain wall, casework). Reviewed for conformance to design intent (not "approved" — that's the contractor's job).
- **Product data** — manufacturer's cut sheets for specified products.
- **Samples** — physical samples (finishes, masonry units, paint chips).

Architect's actions:
- "Reviewed" / "No Exception Taken" — proceed
- "Furnish as Corrected" — minor changes; proceed with corrections
- "Revise and Resubmit" — significant issues; redo
- "Rejected" — not acceptable; submit different product

## Owner / Architect / Contractor roles (AIA A201 framework)

```
Owner ──────── pays for the work; agrees to design + price
   │
   ▼
Architect ─── designs; reviews submittals; certifies pay apps;
              interprets contract documents (NOT inspects)
   │
   ▼
Contractor ── means + methods; site safety; coordination of subs;
              schedule + cost responsibility
```

Common misconceptions:
- The Architect does NOT supervise construction. The Contractor does.
- The Architect does NOT inspect. They observe (site visits) and certify pay apps.
- The Contractor is responsible for safety on site. Not the Architect.

## Materials + assemblies cheat sheet

### Walls
- **Interior partition (non-rated)**: 3-5/8" steel stud + 5/8" GWB each side. Total ~5".
- **Demising / corridor (1HR)**: 3-5/8" steel stud + 5/8" Type X GWB each side. Same as above with Type X.
- **Demising (1HR + STC-50)**: 3-5/8" stud + RC channel + 5/8" Type X + insulation in cavity.
- **Shaft wall (1-2HR)**: Special CH-stud + 1" gypsum core liner + 5/8" Type X.

### Slabs
- **Slab on grade**: 4" concrete + vapor barrier + 4" gravel + compacted subgrade.
- **Suspended slab (post-tensioned)**: typical 7-9" thick spanning column-to-column.
- **Composite metal deck**: 1.5" or 3" steel deck + 4.5" concrete topping.

### Roofs
- **Low-slope membrane (TPO/EPDM/PVC)**: membrane + ISO insulation + cover board + structure.
- **Standing seam metal**: panels on Z-purlins or solid sheathing.

### Curtain wall vs window wall
- **Curtain wall**: structural mullions span floor-to-floor; not load-bearing on slab.
- **Window wall**: punched openings or strip windows; sits on slab edge.

## Reading a set — quick orientation

| Sheet | What you should see |
|---|---|
| A-001 | Cover, project info, sheet index, vicinity map |
| A-002 | Code analysis: use group, construction type, occ load, egress |
| A-101 | Floor plan (one per level) |
| A-201 | Elevations (typically 4) |
| A-301 | Building sections (typically 2-4) |
| A-401 | Enlarged plans (bathrooms, kitchens, etc.) |
| A-501 | Details (door, window, wall sections) |
| A-601 | Schedules (door, window, finish, hardware) |
| S- | Structural |
| M- | Mechanical / HVAC |
| P- | Plumbing |
| E- | Electrical |
| FP- | Fire protection / sprinkler |
| FA- | Fire alarm |

If you can't find a thing in the CD set, it's probably on the
discipline you didn't think to check.

## Substantial completion

- Owner can occupy + use the building for its intended purpose.
- Warranty period starts.
- Final pay app minus retainage.
- Punch list: incomplete items + minor defects.
- Most contracts: SC is the legal trigger for time-related claims.

Punch list closeout typically 30-60 days. Final completion follows.
